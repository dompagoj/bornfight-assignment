## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Testing the app

```bash
Access /?army1=50&army2=75 on port 3000 (default, can be changed with PORT env variable)
where army1 and army2 are arbitrary numbers greater than 0
```
